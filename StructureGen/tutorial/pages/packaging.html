Package Structure - Packages the current structure into a zip file.
Import Structure - Imports a structure from a zip archive from the "StructureGen/packages" folder. Import Tileset - Adds the tiles and/or objects from the tileset to the program. Prior to importing, tilesets are located in the "packages/tilesets/" directory. Requires a restart to take effect. Remove Tileset - Removes the tiles and/or objects that are in the tileset from the program. Requires a restart to take effect.
