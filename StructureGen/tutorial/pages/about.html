Mod Version: Displays the current version of the mod, which is also displayed on the title of the program.

Java Version: Displays the Java version you are running.

Memory Usage:
Available memory - the total memory available to the JVM.
Allocated memory - The amount of memory the JVM has actually given itself.
Memory in use - How much memory the program is actually using.

Forum Thread: Opens the mod's thread on the Starbound forums in the default browser.

Mod Download: Opens the mod's Mediafire download page.
